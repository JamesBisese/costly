Using the Raleigh Cost Tool - Instructions

Setting up your account
Before starting any projects, add information to set up your account by clicking on your username on the far right of the menubar, and then selecting “Account.” Click on the “Edit Account” button and input the information as requested. For information under the “Short Bio” field please input a brief description of your discipline and experience related to site design. For “Desired Level of Tool Use” please select one of the two options under the drop-down menu using the down arrow on the right side of the entry box. Only professionals that will interact with the City of Raleigh regarding project design and review should select the “Professional user” option. All others should select the “Standard user” option. Click on the “Save changes” button and review the summary for accuracy. Edit again if needed, or click on the Project Page and follow the instructions below to set up a new project and begin using the tool.

Setting up a new project
To begin a project site assessment using the Raleigh Cost Tool, navigate to the PROJECTS tab from the menubar and click on the
NEW PROJECT BUTTON.
When you create a new project, specify a title, project description, the location and size of the project, and purchase information. Projects can be exported into .csv, excel, PDF, or printed. The summary table of your project displayed on the PROJECT PAGE can also be copied and pasted into a word document file or spreadsheet.

Designing scenarios for your project
After creating a project, click the
SCENARIOS BUTTON
to view and existing scenarios or create a new one. Click the
NEW SCENARIO BUTTON
to create a new scenario, where you will be directed to a new page to begin adding information about your scenario. You can click to go back to the main SCENARIO PAGE at any point to delete, edit, copy, or export your scenario. When a scenario is selected, the
DUPLICATE SCENARIO BUTTON
will be displayed. When clicked, select yes on the prompt if you want to create a copy of your scenario.

When editing or creating a new scenario, the main COST TOOL PAGE will display five new tabs: (1) Project-Scenario Description, (2) Structure, (3) Structure Costs, (4) Cost Item Unit Costs, and (5) Results. On these tabs, you can click on the green headers (e.g. Project Information) to get more information about the content in that section. Hovering you mouse over individual cells will display help text.
  • The Project-Scenario Description, Non-Conventional Structures, and Conventional Structures Tabs describe features and structures that can occur based on your project type selection. To learn more about each feature or structure and what changes to the Raleigh Unified Development Ordinance apply, simply click on it. Cost information will be generated for both features and structures if you enter a value in the empty cell next to the name. If you don’t want to include a feature or structure in your final cost analysis but want to keep the values you’ve entered in the textbox, you can deselect the feature or structure by clicking on the checkbox.
  • The Structures Cost Tab provides information about the tool’s default assumptions for cost items that are calculated per structure and the project’s life cycle costs. You can update any of the cells with values that are specific to your project scenario. To add or remove cost items, click the checkbox. To view other cost items and/or add cost items to a structure that are not selected as a default cost item, click on the checkbox next to the text ‘Show all Cost Items’.
  • The Cost Item Unit Costs allows for adjustments to the default unit cost for individual cost items. Use the dropdown button in the Unit Cost Column to select values form three choices: (1) RSMeans, (2) the cost tool database, or (3) a user option to define your unit cost information. Default values for the replacement life and ongoing operations and maintenance factor numbers can be changed on this tab as well.
  • The Results Tab provides a summary of the costs for structures in your project site and overall project life cycle costs. Scenario results are saved automatically - if you log out or wish to start a new project or scenario you can review them later.


Comparing scenarios
When at least two scenarios have been created for a project, you can compare the scenarios. To do so, click on the scenarios you want to compare (they will become highlighted) and then click the
COMPARE SCENARIOS BUTTON.
A new page will open displaying summaries of the two scenarios and how they differ from each other. To unselect a scenario, simply click on the row of the scenario you no longer want to have highlighted.

If you experience any problems while using this tool or have any comments about how to improve the tool, you can contact the tool administrator: CostToolAdministrator@raleighnc.gov.
