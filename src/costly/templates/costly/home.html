What is the GSI Cost Tool?

The Raleigh GSI Cost Tool is advancing the implementation of Green Stormwater Infrastructure (GSI) in the City of Raleigh by providing an easy-to-use online platform that educates designers and developers about the new opportunities for GSI locations.



The Cost Tool allows the user to see areas where distributed GSI practices can now occur due to the updates to the Cityâ€™s Unified Development Ordinance (UDO). The tool references a local cost database that is generated by professionals who work in development and design of these practices, along with costs that are being applied to real-world local projects. The purpose is to help the user compare designs using a credible database and to consider localized saving opportunities that are unique to development projects in the Raleigh area.


If you have any questions about how to use the tool, please contact the administrator: CostToolAdministrator@raleighnc.gov
